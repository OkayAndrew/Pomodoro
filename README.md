# Pixel Pomodoro

Pixel Pomodoro is a small, single-page web app that turns the classic Pomodoro technique into a playful pixel-art experience. It combines a visual timer, task entry, customizable work/break intervals, and a simple learning section in one lightweight HTML file.

## What it does

- Runs a Pomodoro timer with work, short break, and long break phases
- Displays a pixel-art character that changes as time progresses
- Lets you enter planned and completed tasks
- Offers adjustable timer settings for work, short break, long break, and cycle count
- Includes a built-in learning panel with Pomodoro basics and tips
- Uses browser notifications and sound cues when phases change

## Project structure

- pomodoro.html — the full app, including HTML, CSS, and JavaScript
- icons/ — icon and favicon assets
- main_images/ — pixel-art character images for different timer states
- sounds/ — audio files for phase transitions

## How to run

1. Open pomodoro.html in a web browser.
2. Start the timer and interact with the icons to view tasks, settings, or learning content.

No build step or installation is required.

## Notes

This project is intentionally simple and self-contained, making it easy to modify or reuse as a personal productivity tool or a prototype for further UI experimentation.
